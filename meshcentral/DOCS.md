This add-on provides the following configuration options:
| **Option** | **Type** | **Default** | **Description** |
| :----------- | :--------: | :-----------: | :---------------- |
| access_domain | str | | Mandatory. For local usage, this can be set to 'localhost', or <your-ip>. For internet access, set this to the domain/sub-domain from which MeshCentral will be accessed. |
| no_of_days_backup | int | | No. days of backups that the autobackup keeps. MeshCentral's default is 10 Days worth |
| backup_interval_hours | int | | No. of hours after which auto backup is taken. MeshCentral's default is 24. |
| backup_files | bool | false | Backup the files & recording folders. MeshCentral's default is false. |
| welcome_text | str | | The text to show at the bottom of the homepage. |

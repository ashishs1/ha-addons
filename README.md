# Home Assistant Add-ons by Ashish

A place to post my own add-ons. You can add this repo to you Homeassistant using this button:

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fashishs1%2Fha-addons)
